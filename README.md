# Blast-Furnace-Monitoring-Database-of-a-Iron-and-Steel-Plant
***
This dataset accompanies the paper Recognition model for blast furnace conditions fusing multimodal information and its industrial application, targeting the scenario of multimodal monitoring and intelligent diagnosis for blast furnace ironmaking. It provides full data for model training, validation, and industrial deployment.

# Start
***
1.Install **Python 3.10**. For convenience, execute the following command.

```
pip install numpy pandas matplotlib scikit-learn torch torchvision tensorflow keras torch_geometric scipy imbalanced-learn opencv-python pillow statsmodels
```

2. Data preparation: The required experimental datasets are available from the above documents and have been fully preprocessed and split.

3. For model training and evaluation, we will upload the experimental codes for all baseline tests later, which can be used to reproduce the experimental results.
